https 全称是Hyper Text Transfer Protocol Secure 相比http 多了一个Secure Secure 是由TLS(SSL)提供的，大概是由OpenSSL的library 提供的，https 和http 都属于application layer ， 基于TCP协议 TCP（传输层） http是普通超文本协议（应用层） 应用层和传输层之间添加SSL协议就是HTTPS ---- HTTPS是安全超文本传输协议 http连接端口是80 https 连接端口是443 http连接很简单， 没有状态 HTTPS 是SSL加密的传输 ，身份认证的网络协议，相对http传输比较安全
